## 워크플로우 확인된 것
- **주문 변환 → 택배송장 출고는 매칭과 무관하게 항상 성공.** 매출도 저장 즉시 정확.

- 즉 담당자가 오전에 수정 못해도 변환·송장·매출엔 문제 없음.
- 몰별 상품명 변형이 생기면 → 매칭데이터 파일에 "수집명 → 올바른 상품명" 추가해서 전달 → 반영.
